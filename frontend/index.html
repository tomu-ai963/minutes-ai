📝 minutes-ai

音声ファイルから議事録を自動生成（BYOK：APIキーはお手元のブラウザに保存されます）

⚙️ 設定

Cloudflare Workers にデプロイした URL を入力してください。

— Notion連携 —

NotionでDBを開き、URLの ?v= の手前にある32桁のIDです。

— Slack連携 —

Slack App 設定の Incoming Webhooks から取得できます。

※ APIキー・設定は このブラウザの localStorage にのみ保存され、サーバーには送信されません（API呼び出し時を除く）。

🎙️ 音声ファイル

クリックして選択 またはドラッグ＆ドロップ
対応形式: mp3 / mp4 / wav / m4a ・ 上限 25MB
